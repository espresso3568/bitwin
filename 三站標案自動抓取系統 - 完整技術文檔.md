## 專案概述

### 🎯 目標

自動化抓取台灣三大研究機構的公開標案資訊，並透過網頁介面即時呈現。

### 📊 資料來源

| 機構 | 網址 | 資料格式 | 抓取方式 |
| :-- | :-- | :-- | :-- |
| 工研院 (ITRI) | vendor.itri.org.tw | JSON API | POST Request |
| 資策會 (III) | bid.iii.org.tw | HTML Table | Web Scraping |
| 中研院 (Sinica) | srp.sinica.edu.tw | HTML Table | pandas.read_html |

### ✨ 核心功能

- ✅ 每天自動執行（台北時間 08:00）
- ✅ 自動合併三站資料
- ✅ 產生 CSV 檔案（含歷史記錄）
- ✅ 產生 JSON API（供前端使用）
- ✅ 即時網頁看板（篩選、搜尋）
- ✅ 完全免費（GitHub 託管）


### 📈 資料流程

```
GitHub Actions (每天 08:00)
    ↓
執行 Python 爬蟲
    ↓
抓取三站資料
    ↓
合併 + 去重
    ↓
輸出 CSV 和 JSON
    ↓
Commit 到 GitHub
    ↓
GitHub Pages 自動更新網頁
```

## 完整程式碼

### 1. scraper/main.py

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
三站標案抓取器 v1.0
===================

功能：
  - 抓取工研院、資策會、中研院標案資訊
  - 輸出 CSV（含歷史）和 JSON（給前端）
  - 自動化執行（GitHub Actions）

作者：AI Assistant
日期：2026/03/05
授權：MIT
"""

import requests
from bs4 import BeautifulSoup
import pandas as pd
from datetime import datetime
import json
import os
import sys

# ============================================================================
# 全域設定
# ============================================================================

TODAY = datetime.now().strftime('%Y%m%d')
TODAY_CN = datetime.now().strftime('%Y-%m-%d')

# 請求 Headers（模擬瀏覽器）
HEADERS = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 '
                  '(KHTML, like Gecko) Chrome/120.0.0.0 Safari/537.36'
}

# 超時設定（秒）
TIMEOUT = 30


# ============================================================================
# 工研院爬蟲
# ============================================================================

def scrape_itri():
    """
    抓取工研院標案（JSON API）
    
    API 端點：
      - Proxy: https://vendor.itri.org.tw/api/JsonRelayHandler.ashx
      - Real: https://abpssapi.itri.org.tw/api/ABPSSAPI/GetpublishDocList
    
    請求參數：
      {
        "BidDocStatus": "",    // 狀態（""=全部, "P"=公告中）
        "IseBid": "",          // 電子標（""=全部, "Y"=電子標）
        "currentPage": 1,      // 頁碼
        "PageSize": 100        // 每頁筆數
      }
    
    回應結構：
      {
        "bddata": [
          {
            "BidInfo": {"CNo": "案號", "CName": "標題", ...},
            "LatestPublishdt": "20260305",
            "EndDate": {"Year": 2026, "Month": 3, "Day": 10, ...},
            "BidDocStatus": "P"
          }
        ]
      }
    
    Returns:
        pandas.DataFrame: 工研院標案資料
    """
    print("📡 抓取工研院...")
    
    url = "https://vendor.itri.org.tw/api/JsonRelayHandler.ashx"
    
    headers = HEADERS.copy()
    headers.update({
        "Accept": "application/json",
        "Content-Type": "application/json",
        "RemoteUrl": "https://abpssapi.itri.org.tw/api/ABPSSAPI/GetpublishDocList",
        "Origin": "https://vendor.itri.org.tw",
        "Referer": "https://vendor.itri.org.tw/broadBqry2.aspx",
    })
    
    payload = {
        "BidDocStatus": "",     # 全部狀態（可改 "P" 只抓公告中）
        "IseBid": "",           # 全部類型（可改 "Y" 只抓電子標）
        "currentPage": 1,       # 第1頁
        "PageSize": 100         # 每頁100筆
    }
    
    try:
        resp = requests.post(url, headers=headers, json=payload, timeout=TIMEOUT)
        resp.raise_for_status()
        data = resp.json()
        
        bids = []
        for row in data.get('bddata', []):
            bid_info = row.get('BidInfo', {})
            end_date = row.get('EndDate', {})
            
            # 拼接截止日期
            end_date_str = ''
            if isinstance(end_date, dict):
                year = end_date.get('Year', '')
                month = str(end_date.get('Month', '')).zfill(2)
                day = str(end_date.get('Day', '')).zfill(2)
                if year and month and day:
                    end_date_str = f"{year}/{month}/{day}"
            
            bids.append({
                '來源': '工研院',
                '案號': bid_info.get('CNo', ''),
                '標題': bid_info.get('CName', ''),
                '採購方式': bid_info.get('BidMethod', ''),
                '是否電子標': bid_info.get('IseBid', ''),
                '公告日': row.get('LatestPublishdt', ''),
                '截止日': end_date_str,
                '狀態': row.get('BidDocStatus', ''),
                '承辦人': bid_info.get('Undertaker', {}).get('name', ''),
            })
        
        df = pd.DataFrame(bids)
        print(f"   ✓ 工研院：{len(df)} 筆")
        return df
    
    except requests.exceptions.Timeout:
        print(f"   ✗ 工研院：連線超時（>{TIMEOUT}秒）")
        return pd.DataFrame()
    except requests.exceptions.RequestException as e:
        print(f"   ✗ 工研院：網路錯誤 - {e}")
        return pd.DataFrame()
    except (KeyError, ValueError) as e:
        print(f"   ✗ 工研院：資料解析錯誤 - {e}")
        return pd.DataFrame()


# ============================================================================
# 資策會爬蟲
# ============================================================================

def scrape_iii():
    """
    抓取資策會標案（HTML GridView 表格）
    
    網址：https://bid.iii.org.tw/bidnewlist.aspx
    
    HTML 結構：
      <table id="GridView1">
        <tr>
          <td>PP26030002</td>         <!-- 案號 -->
          <td>1</td>                  <!-- 訂單 -->
          <td><a>標題</a></td>        <!-- 標題（含連結）-->
          <td></td>                   <!-- 採購類型 -->
          <td>115/03/05</td>          <!-- 公佈日 -->
          <td>115/03/11 17:00</td>    <!-- 投標日 -->
          <td>115/03/11 17:05</td>    <!-- 開標日 -->
        </tr>
      </table>
    
    Returns:
        pandas.DataFrame: 資策會標案資料（前10筆）
    """
    print("📡 抓取資策會...")
    
    url = 'https://bid.iii.org.tw/bidnewlist.aspx'
    
    try:
        resp = requests.get(url, headers=HEADERS, timeout=TIMEOUT)
        resp.raise_for_status()
        soup = BeautifulSoup(resp.text, 'html.parser')
        
        # 找表格
        table = soup.find('table', id='GridView1')
        if not table:
            print("   ✗ 資策會：找不到表格（可能網頁結構改變）")
            return pd.DataFrame()
        
        # 解析表格行（跳過 header，取前10筆）
        rows = table.find('tbody').find_all('tr')[:10]
        bids = []
        
        for row in rows:
            cols = row.find_all('td')
            if len(cols) >= 7:
                link = cols[2].find('a')
                bids.append({
                    '來源': '資策會',
                    '案號': cols[0].text.strip(),
                    '訂單': cols[1].text.strip(),
                    '標題': cols[2].text.strip(),
                    '標題連結': f"https://bid.iii.org.tw{link['href']}" if link else '',
                    '採購類型': cols[3].text.strip(),
                    '公佈日': cols[4].text.strip(),
                    '投標日': cols[5].text.strip(),
                    '開標日': cols[6].text.strip() if len(cols) > 6 else '',
                })
        
        df = pd.DataFrame(bids)
        print(f"   ✓ 資策會：{len(df)} 筆")
        return df
    
    except requests.exceptions.Timeout:
        print(f"   ✗ 資策會：連線超時（>{TIMEOUT}秒）")
        return pd.DataFrame()
    except requests.exceptions.RequestException as e:
        print(f"   ✗ 資策會：網路錯誤 - {e}")
        return pd.DataFrame()
    except Exception as e:
        print(f"   ✗ 資策會：解析錯誤 - {e}")
        return pd.DataFrame()


# ============================================================================
# 中研院爬蟲
# ============================================================================

def scrape_sinica():
    """
    抓取中研院標案（HTML 表格，pandas 自動解析）
    
    網址：https://srp.sinica.edu.tw/InviteBids
    
    查詢參數：
      - searchPubTime: 公告日期（YYYY-MM-DD）
      - searchBidName: 標案名稱（選填）
      - searchInstNo: 機構代碼（選填）
    
    方法：
      使用 pandas.read_html() 自動解析 HTML 表格
    
    Returns:
        pandas.DataFrame: 中研院標案資料（當天公告）
    """
    print("📡 抓取中研院...")
    
    url = f'https://srp.sinica.edu.tw/InviteBids?searchPubTime={TODAY_CN}'
    
    try:
        resp = requests.get(url, headers=HEADERS, timeout=TIMEOUT)
        resp.raise_for_status()
        
        # pandas 自動解析 HTML 表格
        dfs = pd.read_html(resp.text)
        
        if dfs and len(dfs) > 0:
            df = dfs[0]  # 取第1個表格
            df['來源'] = '中研院'
            df['查詢日期'] = TODAY_CN
            print(f"   ✓ 中研院：{len(df)} 筆")
            return df
        else:
            print("   ✗ 中研院：無資料（當天無新公告）")
            return pd.DataFrame()
    
    except requests.exceptions.Timeout:
        print(f"   ✗ 中研院：連線超時（>{TIMEOUT}秒）")
        return pd.DataFrame()
    except requests.exceptions.RequestException as e:
        print(f"   ✗ 中研院：網路錯誤 - {e}")
        return pd.DataFrame()
    except ValueError as e:
        print(f"   ✗ 中研院：表格解析錯誤 - {e}")
        return pd.DataFrame()


# ============================================================================
# 主程式
# ============================================================================

def main():
    """
    主流程：
      1. 抓取三站資料
      2. 合併 DataFrame
      3. 輸出 CSV（歷史 + 最新）
      4. 輸出 JSON（給前端用）
    """
    print("\n" + "="*70)
    print("  三站標案抓取器 v1.0")
    print("="*70)
    print(f"  執行時間：{datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
    print("="*70 + "\n")
    
    # ========================================================================
    # 步驟 1：抓取三站資料
    # ========================================================================
    dfs = [
        scrape_itri(),
        scrape_iii(),
        scrape_sinica(),
    ]
    
    # 過濾空資料
    valid_dfs = [df for df in dfs if not df.empty]
    
    if not valid_dfs:
        print("\n⚠️  警告：所有站點都無資料")
        sys.exit(1)
    
    # ========================================================================
    # 步驟 2：合併資料
    # ========================================================================
    print("\n🔄 合併資料...")
    all_bids = pd.concat(valid_dfs, ignore_index=True)
    print(f"   ✓ 總計：{len(all_bids)} 筆")
```

```python
def scrape_sinica():
    """抓最近7天的公告"""
    from datetime import timedelta
    
    dfs = []
    for i in range(7):  # 往前7天
        date = (datetime.now() - timedelta(days=i)).strftime('%Y-%m-%d')
        url = f'https://srp.sinica.edu.tw/InviteBids?searchPubTime={date}'
        resp = requests.get(url, timeout=TIMEOUT)
        dfs_temp = pd.read_html(resp.text)
        if dfs_temp:

    if dfs:
```

## 疑難排解

### 問題 1：GitHub Actions 執行失敗

**症狀**：Actions 頁籤顯示紅色叉叉

**解決方案**：

1. 點擊失敗的 workflow run
2. 查看錯誤訊息（通常在「Run Scraper」步驟）
3. 常見原因：
    - 網站結構改變 → 更新選擇器
    - 網路超時 → 增加 TIMEOUT 秒數
    - 套件版本衝突 → 更新 requirements.txt

### 問題 2：網頁顯示「資料載入失敗」

**症狀**：網頁顯示錯誤訊息

**解決方案**：

1. 確認 `docs/data.json` 存在且格式正確
2. 用瀏覽器直接訪問 JSON：`https://你的帳號.github.io/bid-scraper/data.json`
3. 檢查 CORS 設定（GitHub Pages 預設允許）

### 問題 3：資料不更新

**症狀**：網頁顯示舊資料

**解決方案**：

1. 清除瀏覽器快取（Ctrl+Shift+R）
2. 檢查 GitHub Actions 是否正常執行
3. 確認 Git commit 有成功（查看 repo 最新提交時間）

### 問題 4：工研院 API 回傳空資料

**症狀**：工研院筆數為 0

**解決方案**：

1. 手動訪問 API 測試：

```bash
curl -X POST https://vendor.itri.org.tw/api/JsonRelayHandler.ashx \
  -H "Content-Type: application/json" \
  -H "RemoteUrl: https://abpssapi.itri.org.tw/api/ABPSSAPI/GetpublishDocList" \
  -d '{"BidDocStatus":"","IseBid":"","currentPage":1,"PageSize":100}'
```

2. 檢查是否 IP 被封鎖（GitHub Actions IP 可能變動）
3. 嘗試加入延遲：`time.sleep(2)`

### 問題 5：中研院無資料

**症狀**：中研院筆數為 0

**原因**：當天無新公告（正常情況）

**解決方案**：

- 改抓最近7天（參考進階設定 \#2）
- 或接受某些日子無資料

```python
# 指定 encoding
dfs = pd.read_html(resp.text, encoding='utf-8')

soup = BeautifulSoup(resp.text, 'lxml')
```
